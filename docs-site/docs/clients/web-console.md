# Web console

No install needed. Open `http://<server-host>:8080/admin` in any modern
browser. On first server run, this is where you create the administrator
account (see [First-run wizard](/getting-started/first-run-wizard)).

The web console covers administration, live view, playback, clips, and
export, it's the fastest way to confirm your server works before
installing anything native, and for many operators it's all they ever
need. It's the most production-ready client Crumb has today.

If you've set up [TLS](/configuration/tls), the same console is also
available at `https://<server-host>:8443/admin`, with the expected
self-signed certificate warning on a LAN install with no public domain.
